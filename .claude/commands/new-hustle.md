## Step 2: Evaluate and pick ONE opportunity
Score each idea on:
- **Speed to first $**: How fast can we earn? (hours > days > weeks)
- **Effort**: Can Claude build/execute it autonomously?
- **Scalability**: Does it compound over time?
- **Capital needed**: We have limited USDC, prefer $0 startup cost

## Step 3: Build the infrastructure
- Create a new slash command in `.claude/commands/<hustle-name>.md`
- Create a tracking file in `memory/<hustle-name>.md`
- Write any scripts needed in `agents/` or a new directory
- Update CLAUDE.md with the new revenue stream

## Step 4: Execute immediately
Don't just plan — actually do the thing:
- If it's content, write and publish it NOW
- If it's a tool, build and ship it NOW
- If it's a service, set it up NOW

## Step 5: Log results
Update memory files with:
- What was created
- Expected revenue timeline
- What to do next to grow it
